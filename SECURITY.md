# سیاست امنیتی

## گزارش مشکلات امنیتی

اگر یک مشکل امنیتی را کشف کردید، لطفاً آن را به صورت عمومی منتشر نکنید. درعوض:

1. به آدرس [security advisory](https://github.com/Aixariaa/Sam/security/advisories) مراجعه کنید
2. یک "Draft Security Advisory" ایجاد کنید
3. جزئیات مشکل را شرح دهید
